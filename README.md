# Dilu

Launch products without writing code. Pick a template, connect your tools, set a domain, ship.

**Live:** [dilu.wokspec.org](https://dilu.wokspec.org)

---

## What it is

Dilu is a no-code product launcher built on WokSpec infrastructure. It handles the parts people usually get stuck on — connecting services, configuring domains, managing deployments — so you can focus on the product itself.

Templates are real, production-ready builds. Not starters. Not boilerplates. Things you can actually ship.

---

| Template | Built for |

| Creator store | Digital products, subscriptions, membership |

## Integrations

Connect what you already use. No API keys required — OAuth flows handle authentication.

| Category | Services |
|----------|----------|
| Payments | Stripe, Lemon Squeezy |
| E-commerce | Shopify |
| Community | Discord |
| Email | Resend, Mailchimp |
| CMS | Notion, Airtable |
| Auth | GitHub, Google |
| Analytics | Plausible, Fathom |

---

## How it works

1. Pick a template
2. Connect integrations (OAuth — no manual key setup)
3. Set a domain (or use a free `*.dilu.app` subdomain)
4. Launch

No config files. No deployment commands. No infrastructure to manage.
